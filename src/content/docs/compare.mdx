## Core execution

| Feature | JamJet | LangGraph | CrewAI | AutoGen |
|---|---|---|---|---|
| Graph-based workflow | <Y/> | <Y/> | <P/> Sequential/hierarchical | <Y/> |
| Async execution | <Y/> | <Y/> | <Y/> | <Y/> |
| Local in-process runner | <Y/> | <Y/> | <Y/> | <Y/> |
| Typed state | <Y/> Pydantic | <P/> TypedDict | <N/> Dict | <P/> Dict |
| State validation at every step | <Y/> | <N/> | <N/> | <N/> |
| Conditional routing | <Y/> Inline predicates | <Y/> Edge functions | <P/> Process type | <Y/> |
| Parallel branches | <Y/> `type: parallel` | <Y/> | <N/> | <Y/> |
| Cycle / loop support | <Y/> | <Y/> | <P/> | <Y/> |

## Durability & reliability

| Feature | JamJet | LangGraph | CrewAI | AutoGen |
|---|---|---|---|---|
| Durable execution (crash recovery) | <Y/> Rust runtime | <E/> Checkpointers | <N/> | <N/> |
| Event sourcing | <Y/> Native | <N/> | <N/> | <N/> |
| Automatic retry with backoff | <Y/> YAML config | <E/> Manual | <E/> Manual | <E/> Manual |
| Human-in-the-loop / pause | <Y/> `type: wait` | <Y/> `interrupt_before` | <N/> | <P/> |
| Resume from any checkpoint | <Y/> | <E/> Requires saver | <N/> | <N/> |
| Timeout per step | <Y/> | <P/> | <N/> | <P/> |
