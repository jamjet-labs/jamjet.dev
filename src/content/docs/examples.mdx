## Hello Agent

The minimal JamJet workflow — one model node, one answer. Start here.

```yaml
workflow:
  id: hello-agent
  version: 0.1.0
  state_schema:
    query: str
    answer: str
  start: think

nodes:
  think:
    type: model
    model: claude-haiku-4-5-20251001
    prompt: "Answer clearly: {{ state.query }}"
    output_key: answer
    next: end

  end:
    type: end
```

```bash
jamjet run workflow.yaml --input '{"query": "What is JamJet?"}'
```

## Research Agent

Searches the web with Brave Search (MCP) and synthesizes a structured report.

```yaml
nodes:
  search:
    type: tool
    server: brave-search
    tool: web_search
    arguments:
      query: "{{ state.query }}"
      count: 10
    output_key: results
    next: synthesize

  synthesize:
    type: model
    model: claude-sonnet-4-6
    prompt: |
      Results: {{ state.results }}
      Write a report on: {{ state.query }}
    output_key: report
    next: end
```

Requires `BRAVE_API_KEY`. See [MCP docs](/docs/mcp) for setup.

## Code Reviewer

Fetches a GitHub PR via MCP, reviews it with Claude, scores quality with an eval node, then posts a PR comment. Retries with feedback if quality is low.

```yaml
nodes:
  fetch-pr:
    type: tool
    server: github
    tool: get_pull_request
    arguments:
      repo: "{{ state.repo }}"
      number: "{{ state.pr_number }}"
    output_key: pr_data
    next: review

  review:
    type: model
    model: claude-sonnet-4-6
    prompt: "Review this PR: {{ state.pr_data }}"
    output_key: review
    next: check-quality

  check-quality:
    type: eval
    scorers:
      - type: llm_judge
        rubric: "Is this a thorough, constructive code review?"
        min_score: 4
    on_fail: retry_with_feedback
    next: post-comment
```

## Approval Workflow

Agent proposes a plan, pauses for human approval, then executes or handles rejection. 24-hour timeout with escalation.

```yaml
nodes:
  propose:
    type: model
    model: claude-sonnet-4-6
    prompt: "Propose a plan for: {{ state.task }}"
    output_key: proposal
    next: await-approval

  await-approval:
    type: wait
    event: human_approved
    timeout_hours: 24
    on_timeout: escalate
    next: execute
```

Resume from another terminal:

```bash
jamjet resume <exec-id> --event human_approved --data '{"approved": true}'
```

## Orchestrator + Specialists

Orchestrates specialist agents (research, writing, fact-check) via A2A. Runs research and outline in parallel.

```yaml
nodes:
  parallel-gather:
    type: parallel
    branches:
      - research
      - outline
    join: draft

  research:
    type: a2a_task
    agent_url: "{{ env.RESEARCH_AGENT_URL }}"
    input:
      query: "{{ state.plan.research_query }}"
    output_key: research

  draft:
    type: a2a_task
    agent_url: "{{ env.WRITER_AGENT_URL }}"
    input:
      outline: "{{ state.outline }}"
      research: "{{ state.research }}"
    output_key: draft
    next: end
```

## Customer Support Bot

Python SDK workflow — classifies tickets with Haiku, searches a knowledge base via MCP, drafts replies with Sonnet.

```python
@workflow(id="support-bot", version="0.1.0")
class SupportBot:
    @node(start=True)
    async def classify(self, state: State) -> State:
        response = await self.model(
            model="claude-haiku-4-5-20251001",
            prompt=f"Classify this ticket: {state['ticket']}",
            system="Output JSON: {category, priority}",
        )
        return {"classification": response.json()}

    @node
    async def draft_reply(self, state: State) -> State:
        response = await self.model(
            model="claude-sonnet-4-6",
            prompt=f"Draft a support reply: {state['kb_results']}",
        )
        return {"reply": response.text}
```

## Data Pipeline

Durable ETL: fetches from 3 APIs in parallel (with retry), transforms with Claude, stores the result. Crash-safe — resumes from last checkpoint.

```yaml
nodes:
  extract:
    type: parallel
    branches: [fetch-api1, fetch-api2, fetch-api3]
    join: transform

  fetch-api1:
    type: http
    method: GET
    url: "{{ env.API1_URL }}/data?date={{ state.date }}"
    output_key: raw_api1
    retry:
      max_attempts: 3
      backoff: exponential
```

## SQL Agent

Ask your database questions in plain English. Generates SQL with Claude, validates it for safety, executes via PostgreSQL MCP.

```yaml
nodes:
  generate-sql:
    type: model
    model: claude-sonnet-4-6
    system: "Generate safe SELECT-only SQL. Output JSON: {sql, explanation}."
    prompt: |
      Schema: {{ state.schema }}
      Question: {{ state.question }}
    output_key: sql_response
    next: validate-sql

  validate-sql:
    type: branch
    conditions:
      - if: "'SELECT' in state.sql_response.sql.upper()"
        next: run-query
    default: reject-sql
```
